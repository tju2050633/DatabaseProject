# 7.3

- **Vue前端开发环境配置**
  - `npm 9.5.1`
  - `node v18.16.0`
- **编写.vue文件**
  - `template` 内编写 `html` 代码
  - `script` 内编写 `javascript` 代码
  - `style` 内编写 `css` 代码
- **熟悉组件间的交互逻辑**
  - 父组件 -> 子组件：`props`
  - 子组件 -> 父组件：`event`
  - 例子：
    - 帖子编辑部分 -> `event` -> 用户动态页面 -> `props` -> 帖子内容部分
- **创建示范项目(类似QQ空间)**
  - 页面
    - 主页
    - 登录页面
    - 注册页面
    - 用户动态页面
      - 用户信息部分
      - 帖子编辑部分
      - 帖子内容部分
    - 好友列表页面
    - 404页面
  - 页面结构
    - 导航栏
      - 路由实现页面跳转
    - 内容框
